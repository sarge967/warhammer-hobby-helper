Test 2 — Is the GEMINI_API_KEY environment variable set?
Not run yet
The API key isn't set. To fix:
1. Go to your Netlify site → Site configuration → Environment variables
2. Click Add a variable
3. Key: GEMINI_API_KEY   Value: your key from aistudio.google.com/app/apikey
4. Save, then go to Deploys → Trigger deploy → Deploy site
Test 3 — Does the Gemini API key actually work?
Not run yet
The API key was rejected by Google. To fix:
• Go to aistudio.google.com/app/apikey and create a new key
• Make sure you copied the full key with no extra spaces
• Update the GEMINI_API_KEY environment variable in Netlify with the new key
• Redeploy after changing it
